Pregunta 36 de 50

Su organización está desarrollando una aplicación orientada al cliente que usa Azure OpenAI en Foundry Models para generar respuestas personalizadas. La aplicación está conectada a un índice de Microsoft Azure AI Search.

Debe configurar la aplicación para recuperar los datos pertinentes del índice de búsqueda.

¿Qué tiene que hacer?

Seleccione solo una respuesta.

Implemente un modelo de Azure OpenAI adicional.


Habilite la búsqueda semántica.

Esta respuesta es correcta.

Aumente el tamaño del fragmento para la ingesta de datos.


Use la búsqueda de palabras clave.

La habilitación de la búsqueda semántica mejora la precisión y relevancia de los resultados de búsqueda mediante la interpretación del significado detrás de los términos de consulta, lo que lo convierte en la solución más adecuada para mejorar la precisión de la respuesta en este escenario. El aumento del tamaño del fragmento afecta al procesamiento de datos, pero no afecta directamente a la precisión de los datos recuperados. La implementación de un modelo de Azure OpenAI adicional no aborda la necesidad específica de mejorar la precisión de recuperación de datos del índice de búsqueda. El uso de la búsqueda de palabras clave carece de funcionalidades semánticas, que son esenciales para lograr el nivel de precisión deseado.
