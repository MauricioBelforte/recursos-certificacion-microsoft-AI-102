# Recursos para la Certificación AI-102: Designing and Implementing a Microsoft Azure AI Solution

¡Bienvenido/a! Este repositorio está dedicado a recopilar y organizar recursos, notas de estudio, ejemplos de código y todo lo necesario para prepararse y aprobar el examen de certificación **Microsoft AI-102: Designing and Implementing a Microsoft Azure AI Solution**.

## 🎯 Sobre la Certificación AI-102

La certificación **Azure AI Engineer Associate** valida las habilidades para construir, gestionar e implementar soluciones de IA que aprovechan los servicios de Azure AI. [2, 8] Un Ingeniero de IA de Azure trabaja con arquitectos de soluciones, científicos de datos y otros desarrolladores para crear soluciones completas y seguras que integran capacidades de inteligencia artificial. [5]

Los candidatos deben tener experiencia en el desarrollo de soluciones utilizando lenguajes como Python o C# y ser capaces de usar SDKs y APIs REST para construir soluciones en áreas como procesamiento de imágenes, lenguaje natural, minería de conocimiento e IA generativa. [5]

## 📚 Habilidades Medidas en el Examen

El examen AI-102 evalúa tu capacidad para llevar a cabo las siguientes tareas técnicas. Los porcentajes indican el peso relativo de cada área en el examen. [1, 7]

*   **Planear y administrar una solución de Azure AI (20-25%)**: Incluye la selección de los servicios adecuados, la planificación y configuración de la seguridad, y la gestión y monitoreo de los servicios. [1, 13]
*   **Implementar soluciones de IA generativa (15-20%)**: Se enfoca en la creación de soluciones con Azure AI Studio y Azure OpenAI, optimización de modelos y desarrollo de agentes. [11, 13]
*   **Implementar una solución agéntica (5-10%)**: Creación y personalización de agentes de IA. [13]
*   **Implementar soluciones de Computer Vision (10-15%)**: Análisis de imágenes y video, e implementación de modelos de visión personalizados. [2, 13]
*   **Implementar soluciones de Procesamiento de Lenguaje Natural (PLN) (15-20%)**: Análisis y traducción de texto, procesamiento de voz y creación de modelos de lenguaje personalizados. [1, 13]
*   **Implementar soluciones de minería de conocimiento y extracción de información (15-20%)**: Uso de Azure AI Search e Inteligencia de Documentos para extraer información. [1, 13]
